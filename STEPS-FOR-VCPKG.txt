
An x64 build is necessary because libmysql does not support a 32-bit build.

Create directory C:\src

Installation of vcpkg: open CMD window and run the following commands:
  cd C:\src
  git clone https://github.com/Microsoft/vcpkg.git
  .\vcpkg\bootstrap-vcpkg.bat

Installation of PMP dependencies: open CMD window and run the following commands:
  cd C:\src\vcpkg
  vcpkg install taglib --triplet x64-windows
  vcpkg install qt5-base[mysqlplugin] --triplet x64-windows
  vcpkg install qt5[essentials] --triplet x64-windows

Building PMP with CMake using cmake-gui:
  set CMAKE_BUILD_TYPE to Debug or Release
  set CMAKE_TOOLCHAIN_FILE to C:\src\vcpkg\scripts\buildsystems\vcpkg.cmake
  set VCPKG_TARGET_TRIPLET to x64-windows
  configure, set to Visual Studio, platform x64
  generate
